Short Note on Web Technology

Web technology refers to the means by which computers

communicate with each other using markup languages and multimedia packages.

It gives us a way to interact with hosted information,

like websites. Web technology involves
the use of hypertext markup language (HTML) and cascading style sheets (CSS).

According to Microsoft, web technologies include the following: [BOLD TEXT]

1) Mark-up languages, such as HTML, CSS, XML, CGI, and HTTP (Front-end or Client-side technologies). [ITALICS TAG]

2) Programming languages and technologies that help create applications for the web. Some of these are Perl, C#, Java, Visual Basic, and .NET (Back-end or Server-side technologies). [UNDERLINE TAG]

3) Business applications that are customized for specific execution of tasks on a network. [STRIKE TAG]

4) Databases, which are extremely important for data and information storage on a computer network. [SUBSCRIPT TAG]

5) Web server and server technologies that enable request handling on a network, where different users have to share the same resources and communicate with each other. [SUPER SCRIPT TAG]
